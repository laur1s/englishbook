# Unit 19: Technology and Communication

## Learning Objectives / Mokymosi tikslai

- Explain a simple technical process.
- Form common adverbs of manner.
- Write a clear message and follow basic online-safety advice.

## Language Focus / Kalbos pagrindai

Many adverbs use adjective + **-ly**: *careful → carefully*. Common irregular forms include *good → well* and *fast → fast*. Use **first, next, then, finally** to order instructions.

<p class="lt-text" lang="lt"><strong>Lietuviška atrama:</strong> Būdo prieveiksmis atsako į klausimą „kaip?“: <strong>carefully, clearly</strong>. Išimtys: <strong>good → well</strong>, <strong>fast → fast</strong>. Instrukcijas tvarkykite žodžiais <strong>first, next, then, finally</strong>.</p>

## Exercise 1: Match Technology Words / Pratimas 1

1. attach 2. download 3. password 4. screen 5. settings

a. secret access word; b. options you can change; c. add a file to a message; d. get a file from the internet; e. display on a device

## Exercise 2: Form the Adverb / Pratimas 2

1. Please type _____ (careful). 2. The app works _____ (good). 3. He replied _____ (quick). 4. Speak _____ (clear). 5. The file downloaded _____ (slow). 6. This computer runs _____ (fast).

## Exercise 3: Sequence the Instructions / Pratimas 3

Put these steps in one safe, logical order: **choose the file; open a new message; check the recipient; attach the file; press send**. Add **first, next, then, finally**. The recipient must be checked before you press send.

## Exercise 4: Read a Support Chat / Pratimas 4

> User: I can't join the video call. The screen is black. Support: First, check that the camera is on. Next, close other apps that use the camera. Then leave the call and join again. User: It works now, but people can't hear me. Support: Open settings and choose the correct microphone.

1. What was the first problem?
2. What should the user close?
3. Did joining again solve the camera problem?
4. Which setting should the user check next?

## Exercise 5: Spot the Unsafe Action / Pratimas 5

Choose and correct the unsafe action: **use a different password for important accounts; check links carefully; send your password to a stranger; update apps regularly**.

## Exercise 6: Write Helpful Instructions / Pratimas 6

Explain how to send a photo or join a call in 5–7 steps. Model: **First, open the app. Next, choose the person carefully. Then press the camera icon and select a photo. Check the photo before you send it. Finally, press send.**
