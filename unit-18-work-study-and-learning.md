# Unit 18: Work, Study, and Learning

## Learning Objectives / Mokymosi tikslai

- Describe responsibilities and learning goals.
- Use common **verb + to** and **verb + -ing** patterns.
- Ask for clarification in a class or workplace.

## Language Focus / Kalbos pagrindai

Common verbs followed by **to + verb** include **want, need, hope, plan, decide**. Common verbs followed by **-ing** include **enjoy, finish, avoid, practise**. **Like/love/hate** can often use either form with little difference at this level.

<p class="lt-text" lang="lt"><strong>Lietuviška atrama:</strong> Po <strong>want, need, hope, plan, decide</strong> dažniausiai vartojama <strong>to + veiksmažodis</strong>. Po <strong>enjoy, finish, avoid, practise</strong> vartojama <strong>-ing</strong> forma. Mokykitės visą junginį, ne vien atskirą veiksmažodį.</p>

1. receptionist 2. mechanic 3. designer 4. teaching assistant 5. delivery driver

a. supports learners; b. repairs vehicles; c. welcomes visitors; d. creates visual work; e. takes parcels to customers

## Exercise 2: Choose the Verb Pattern / Pratimas 2

1. I hope **to start / starting** a course.
2. She enjoys **to work / working** with children.
3. We finished **to write / writing** the report.
4. They need **to learn / learning** new software.
5. He avoids **to drive / driving** at night.
6. Do you plan **to apply / applying** for the job?

## Exercise 3: Categorise the Verbs / Pratimas 3

Put **decide, enjoy, practise, want, finish, hope** under **to + verb** or **verb + -ing**. Then make one sentence from each group.

## Exercise 4: Read a Course Email / Pratimas 4

> Hello Rasa, Your evening computer course starts on 4 September. Classes run on Tuesdays and Thursdays from 18:00 to 19:30. Please bring a laptop. You need to install the free program before the first class. If you have trouble installing it, reply to this email. We look forward to meeting you.

1. When does the course start?
2. How often are the classes?
3. What must Rasa bring?
4. What should she do if installation is difficult?

## Exercise 5: Clarify a Task / Pratimas 5

a. Certainly. First, open the shared folder. b. Sorry, could you explain the first step again? c. Thanks. Do I need to rename the file? d. Yes, please use today's date.

## Exercise 6: Make a Learning Plan / Pratimas 6

Write six sentences with two goals, two activities you enjoy, one learning need, and one request for help. Model: **I want to improve my computer skills. I plan to take an evening course. I enjoy learning with other people. I enjoy practising at home. I need to understand spreadsheets. Could you show me how to make a chart?**
