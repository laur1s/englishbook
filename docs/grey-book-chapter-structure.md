# Grey's Book Canonical Chapter Structure

## Purpose

This document defines the target lesson shape for every Grey's Book chapter page so later cleanup tasks normalize content toward one stable reading experience.

## Canonical section order

1. English chapter title
2. Lithuanian chapter title
3. `Phrasal Verbs`
4. `Word Bank`
5. `Story`
6. `Understanding Questions`
7. `Creative Questions`
8. `Fun Activities`
9. `Grammar Practice`

## Section rules

- `Phrasal Verbs` is required.
- `Word Bank` is required and should appear only once per chapter.
- `Story` is required and always comes before any practice block.
- `Understanding Questions` is required.
- `Creative Questions` is required.
- `Fun Activities` is expected for the full lesson shape and should group any numbered activities under one heading.
- `Grammar Practice` is expected for the full lesson shape and should use this exact heading.
- Do not use `Grammar Exercises` as a separate naming pattern.
- Do not repeat `Word Bank` at the end of the chapter unless it is intentionally renamed to a different support section.

## Naming conventions

- English and Lithuanian chapter titles should both use Arabic chapter numbering.
- Section headings should use one stable English label, with Lithuanian help text inline only when needed.
- Activity numbering should be sequential inside `Fun Activities`.
- If a chapter has no answerable activity, the activity should not be listed in the answer key.

## Reader contract

- The chapter page should expose sections in the same order learners are expected to complete them.
- The table of contents should map to one clear section per heading with no duplicate destination names.
- Answer preview content should only reflect sections that exist on the visible chapter page.
- Previous/next navigation should assume this section order stays stable across chapters.

## Answer-key contract

- Each chapter answer bundle should align to the visible chapter structure:
  - `Understanding`
  - `Fun Activities`
  - `Grammar Practice`
- If a chapter omits `Fun Activities` or `Grammar Practice`, the shared answer key must omit them too.
- If a chapter includes seven numbered activities, the answer key should either cover all answerable activities or clearly indicate which are discussion-only or model-answer tasks.

## Chapter normalization priorities

- Use Chapter 1 and Chapter 5 as the strongest current structure references.
- Fix order defects first.
- Fix duplicate section names second.
- Restore missing lesson blocks before changing visual reader behavior.
- Normalize answer-key naming immediately when chapter naming changes.
