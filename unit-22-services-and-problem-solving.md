# Unit 22: Services and Problem-Solving

## Learning Objectives / Mokymosi tikslai

- Explain a problem with a product or service.
- Use **too + adjective** and **adjective + enough**.
- Request a repair, replacement, or refund politely.

## Language Focus / Kalbos pagrindai

Use **too** before an adjective for more than acceptable: *The shoes are too small.* Use **enough** after an adjective: *They aren't wide enough.* Present Perfect can describe a current result: *The screen has stopped working.*

<p class="lt-text" lang="lt"><strong>Lietuviška atrama:</strong> <strong>Too</strong> eina prieš būdvardį ir reiškia „per daug“: <strong>too small</strong>. <strong>Enough</strong> eina po būdvardžio ir reiškia „pakankamai“: <strong>wide enough</strong>. Skundą pradėkite faktais, tada mandagiai paprašykite sprendimo.</p>

## Exercise 1: Match Problems and Solutions / Pratimas 1

1. wrong size 2. damaged parcel 3. late delivery 4. missing part 5. faulty screen

a. replacement part; b. exchange; c. repair; d. delivery investigation; e. damage report

## Exercise 2: Choose Too or Enough / Pratimas 2

1. This jacket is **too / enough** tight.
2. The box isn't strong **too / enough**.
3. The room is **too / enough** cold to work in.
4. Is the cable long **too / enough**?
5. These instructions are **too / enough** confusing.
6. The table is big **too / enough** for six people.

## Exercise 3: Build Polite Requests / Pratimas 3

Reorder.

1. could / replace / you / it / please
2. like / refund / I'd / a
3. problem / help / could / this / with / you
4. receipt / here / my / is

## Exercise 4: Read a Return Conversation / Pratimas 4

> Customer: I bought these headphones on Monday, but the left side has stopped working. Assistant: I'm sorry about that. Do you have the receipt? Customer: Yes. Could you replace them, please? Assistant: Certainly. We don't have the same colour, but you can choose the black ones or receive a refund.

1. When were the headphones bought?
2. What has happened?
3. What does the customer request first?
4. Which two solutions are available?

## Exercise 5: Improve the Complaint / Pratimas 5

Rewrite politely: **This is bad. Give me money. It doesn't work.** Include what you bought, the problem, evidence, and your preferred solution.

## Exercise 6: Role-Play a Service Problem / Pratimas 6

Model: **I bought this lamp last week, but it isn't bright enough and the switch has broken. Here is my receipt. Could you replace it, please? If that isn't possible, I'd like a refund.** Use a calm greeting, facts, and a clear request.
